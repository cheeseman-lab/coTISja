# notebooks

This is a directory containing analytical notebooks and ad-hoc code for rapid prototyping. Its contents and key outputs from are:
- `differential_tis.ipynb`: Prototyping of differential analysis for TIS usage and expression across all samples. The same processes have been copied to `scripts/run_differential_analysis.py`
    - Depends on: the formatted `tisdiff` inputs from `filter_ribotish.ipynb`
    - Outputs: Reformatted input metadata and matrices prepared for DESeq2 (`input_sample_metadata.csv`, replicate-level `input_counts_matrix.csv`, condition-level `rna_summed_replicate_counts.csv` and `rpf_summed_replicate_counts.csv`), tables of summary statistics from DESeq for each pairwise comparison (`pairwise_wald`, `pairwise_wald_rna`), fitted translation efficiency values, with (`translation_efficiency_vst_matrix`) and without (`log_translation_efficiency_coeffs.csv`) a variance stabilizing transformation, and results from a likelihood ratio test for whether there is any difference accross samples for a given TIS (`lrt_results.csv`). Results are saved to `data/tisdiff_results`
- `explore_tis_data.ipynb`: Outdated and likely defunct, prototyping some global analyses to check the quality of the filtering procedure on ribotish outputs
- `figures.ipynb`: Generation of figures for my rotation project presentation
- `filter_ribotish.ipynb`: Prototyping of ribotish result filtering procedure. The final process has been implemented in `scripts/filter_ribotish.py`
    - Outputs: Manifest files that are used to direct sbatch job arrays are created in the section titled "Prepare input data and make manifest files" (`ribotish_replicate_manifest.csv`, `ribotish_sample_manifest.csv`, `ribotish_tisdiff_manifest.tsv`), along with preparation of the input files to `ribotish tisdiff`, which are saved to `data/tisdiff_inputs`
- `leaky_scanning_model.ipynb`: Prototyping of the generalized linear model used to model the effects of mechanistic factors on translation efficiency at canonical start sites. The same analysis has been copied to `scripts/fit_leaky_scanning_model.py`. Also reformats the inputs and tests out some potential extensions to the model, such as adding GC content as a predictor and using a random forest model as an alternative prediction method. Future modifications to this model should be careful not to encode redundant predictors (i.e. each feature should be as independent as possible) or else to ensure that all dependent interactions are specified by the model formula. For example, it is probably unwise to include the number of {upstream AUGs, upstream non-AUGs, upstream extensions, and upstream uORFs} as a set of predictors, since the codon variables are dependent on the frame variables and vice versa.
    - Depends on: the compiled TIS tables from `tis_gene_analysis.ipynb` (and optionally the QC-masked readcount matrices from `scripts/run_differential_analysis.py`)
    - Outputs: The formatted input data per transcript with predictors as columns (`glm_model_inputs`), a summary of parameter estimates and model fit (`canonical_translation_model_summary.csv`), the estimated covariance between predictors (`canonical_translation_covariance_matrix.csv`), and a table of annotations of TISs per transcript. Files are saved to `data/translation_models`
- `prepare_igv_files.ipynb`: Ad-hoc generation of IGV tracks based on filtered Ribotish results
    - Outputs: .wig files saved to `data/igv_files`
- `sample_comparison.ipynb`: Prototyping of all major analyses directly comparing TISs or transcripts between samples. Generates summary statistics on the pairwise comparisons between samples, performs functional enrichment of SLiMs on the differential extensions or truncations, runs an analysis of outlier detection for each TIS, runs vector comparisons between samples based on the translation at each TIS along a transcript, performs standard pairwise DESeq2 downstream analysis such as making volcano plots and running GO term enrichment - specifically for the comparison of RPE1 states.
    - Depends on: the compiled TIS tables from `tis_gene_analysis.ipynb` and the differential analysis results from `scripts/run_differential_analysis.py`
    - Outputs: A table of summary statistics at the TIS level (`deseq2_tis_summary_statistics.csv`), further broken into outliers (`alternative_translation_efficiency_outliers.csv`) and continuously varying translation efficiency (`alternative_translation_efficiency_continuous.csv`), an independent annotation of SLiMs in the differential protein sequence for all TISs identified in any sample (`differential_aaseq_slim_analysis.csv`), various reformatting of pairwise statistics (`lfc_matrix_stack.csv`, `pval_matrix_stack.csv`, `qval_matrix_stack.csv`), and transcript-level translation vector summary statistics (`transcript_tis_vector_difference_summary.csv`, `gene_tis_vector_difference_summary.csv`) and raw encodings of their TE values (`transcript_tis_vector_logte_table.csv`, `gene_tis_vector_logte_table.csv`). Results are saved to `data/tisdiff_results`
- `tis_gene_analysis.ipynb`: Prototyping of the appropriate normalization for translation efficiency and analyses about mechanistic factors controlling translation, such as codon identity, UTR length, Kozak sequence context. Notably contains an initial section ("Generate saved tables") which combines results from all samples after filtering ribotish outputs at the sample and replicate level.
    - Depends on: the results from `scripts/filter_ribotish.py`
    - Outputs: Tables of TISs across all samples with corresponding readcount results and annotations. All TIS are compiled into a few tables with different levels of annotation (`all_samples_filtered.csv`, `all_samples_filtered_with_protein_seq.csv`, `all_samples_mane_only.csv` for TISs mapping to MANE Select transcripts). Alternative TISs are additionally compiled in reference to their canonical start sites (`all_samples_paired_tis.csv`). Tables are saved to `data/filtered_tis_data`.
